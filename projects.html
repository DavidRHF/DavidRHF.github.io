Red team internship · Spring 2026

Bash Bunny payload development & physical access testing

A semester of authorized penetration testing against university lab systems using a Hak5 Bash Bunny Mark II — a USB device that presents itself as a trusted keyboard and types a payload the instant it's plugged in.

  • Wrote keystroke-injection payloads in DuckyScript, with Bash helpers for staging and cleanup.
  • Tested how quickly a payload could reach execution on both locked and unlocked hosts.
  • Watched the defensive side in parallel — reading system and network logs to see which actions produced alerts and which passed silently.
  • Wrote up findings with remediation steps: USB port policy, screen-lock enforcement, and endpoint logging gaps.

Stack

DuckyScriptBash Hak5 Bash Bunny Mk IIWindows LinuxLog analysis

Takeaway

The interesting part wasn't getting the payload to run — it was seeing how much of it never showed up in the logs at all.

Wazuh SIEM deployment & log pipeline

Deployed Wazuh agents across university systems for host-based intrusion detection and centralized log collection, then reconfigured services through JSON and XML config files so events landed in a consistent, parseable shape. Tuning mattered more than installing — an untuned ruleset buries the alerts you care about.
